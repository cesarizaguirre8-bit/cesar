# cesar

Frontend visual del curso de psicotrading con identidad de **Pro Trading Academy USA**.

## Cambios principales
- Se integró la imagen entregada como logo principal (`assets/protrading-logo.svg`).
- El logo se usa en la barra superior y también como imagen visual del bloque principal.
- Estilo visual adaptado a tonos oscuros/azules con acentos dorados para reflejar la marca.
- Se mantiene la sección de clases en vivo y las tarjetas de los 3 módulos.

## Ejecutar
```bash
python3 -m http.server 8000
```
Abrir: `http://127.0.0.1:8000`.
